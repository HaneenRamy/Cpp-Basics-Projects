# Tic-Tac-Toe Game (C++)

## Description
A C++ console-based Tic-Tac-Toe game for two players.  
The project demonstrates basic programming concepts such as arrays, loops, functions, and conditional statements.

## Features
- Two-player gameplay (X and O)
- Clear console-based board display
- Input validation
- Automatic win and tie detection
- Simple and interactive user interface

## Technologies
- C++
- Console input/output
